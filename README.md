# TOR-Correlation-Forensic-System.
A forensic system that captures TOR traffic, correlates it with live relay data, and identifies probable origin IPs for  cyber investigations.
TOR Correlation Forensic System is a cyber-forensics tool designed to capture TOR network traffic, analyze it against live TOR relay metadata, and identify probable origin IPs behind anonymous TOR communications.
This system helps police cyber units trace suspicious TOR-based activities and generate investigation-ready forensic reports.
